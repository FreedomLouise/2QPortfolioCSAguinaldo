What is CSS?

  • Short for Cascading Style Sheets
  • It designs how the HTML will look like
  • Located in the stylesheet/style part

CSS Structure:

  • Selector = what part of the HTML you want to design
    • Example: p { color: hotpink; }

      • Types of Selectors:

      • Element Selector – selects elements by name (e.g. p { color: hotpink; })
      • Universal Selector – selects all elements using *
      • ID Selector – selects ONE element, represented with #
      • Class Selector – selects elements under the same class, represented with .
      • Grouping Selector – groups different elements with the same style (e.g. h1, h3, h4 { text-align:center; })
      • Descendant Selector – targets elements under another element (e.g. div.center p { color: hotpink; })
      • Child Selector – selects direct children only (e.g. body > p { color: hotpink; })
  • Property = what you want to change (e.g. color, font-size, background-color)
  • Value = specific design (e.g. hotpink for color, 12px for font-size)
  • Declaration = combination of property and value (e.g. color: hotpink;)

Types of CSS

  • Inline CSS – in the style attribute; most prioritized
  • Internal/Embedded CSS – in the <style> tag inside <head>; applies within one HTML page
  • External CSS – in a separate .css file linked to HTML

CSS Properties

  • Colors – can be RGB, HEX, HSL, RGBA, HSLA, or named colors.
  • Background – controls the background design.
    • background-color – sets background color.
    • background-image – sets an image as background.
    • background-repeat – controls repetition (no-repeat, repeat-x, repeat-y).
    • background-position – positions the background (center, top right, etc.).
    • background-attachment – controls if it scrolls or stays fixed.

CSS Fonts

  • The font properties define the appearance of text.
  • font-family – defines the typeface (serif, sans-serif, monospace, cursive, fantasy).
  • font-style – normal, italic, or oblique.
  • font-weight – sets the thickness of the text (normal, bold, bolder, lighter).
  • font-size – sets the size of the text (px, em, %, etc.).
  • font-variant – small-caps style for uppercase look.
  • All can be combined in shorthand: font: style weight size family;

CSS Border

  • The border defines how the outline of an element appears.
  • border-color – sets the color of the border. You can target each side individually.
  • border-style – determines the border’s pattern (solid, dotted, dashed, double, etc.).
  • border-width – defines thickness of the border (thin, medium, thick, or px values).
  • All can be combined in shorthand: border: width style color;

CSS Margins

  • Margins create space around elements (outside borders).
  • Can have negative values.
  • Shorthand syntax: margin: top right bottom left;
  • Individual sides:
    • margin-top
    • margin-bottom
    • margin-left
    • margin-right
  • auto value centers the element horizontally.
  • Top/bottom margin doesn’t apply to inline elements.

Box Model

  • The CSS box model is the design foundation of all elements.
  • Each element consists of:
    • Content – the text or image inside
    • Padding – space between content and border
    • Border – wraps around padding and content
    • Margin – space outside the border
<!DOCTYPE html>
<html>
<head>
<style>
div {
  background-color: lightgrey;
  width: 300px;
  border: 15px solid green;
  padding: 50px;
  margin: 20px;
}
</style>
</head>
<body>
<h2>Demonstration of the Box Model</h2>
<div>This is a div element with border, padding, and margin.</div>
</body>
</html>

Pseudo-class Syntax

  • Basic syntax:
selector:pseudo-class {
  property: value;
}
  • Pseudo-classes can be used on any selector.
  • Can combine with classes or elements.
div.content:hover {
  color: #ff0000;
}

Example: Hover to reveal text

<!DOCTYPE html>
<html>
<head>
<style>
p {
  display: none;
  background-color: green;
  padding: 20px;
}
div:hover p {
  display: block;
}
</style>
</head>
<body>
<div>Hover over me to show the p element
  <p>This is a p element</p>
</div>
</body>
</html>

Common Pseudo-classes

  • :active – selects active element
  • :checked – selects checked boxes
  • :disabled – selects disabled elements
  • :focus – selects element in focus (e.g. textbox)
  • :valid – selects elements with valid input

Debugging Techniques in CSS

  • CSS errors don’t stop the page but affect appearance.
  • Tips:
    • Check proper selector, property, and value combinations.
    • Understand the box model for spacing/alignment issues.
    • Use “Inspect Element” in browsers to see applied styles.
    • Use browser developer tools for deeper debugging.
